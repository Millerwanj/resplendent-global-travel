## Before launch
1. Replace the temporary Gmail with approved domain email addresses:
   `info@`, `bookings@`, `corporate@`, `business@`, `support@`, and `accounts@`.
2. Connect the form to a secure form endpoint or CRM.
3. Confirm and insert the production domain everywhere.
4. Add verified social links and privacy terms.
5. Confirm image licensing and create AVIF/WebP variants.
6. Run Lighthouse, accessibility, browser and device QA.
