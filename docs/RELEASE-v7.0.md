# Resplendent Global Travel Solutions v7.0

## Release purpose
A production-engineering refinement of the approved v6.0 website. This release deliberately avoids editorial or design changes.

## Key outcomes
- Professional-domain enquiry routing
- More resilient mobile navigation and keyboard access
- Lower layout shift and more efficient image loading
- Stronger technical SEO and social sharing metadata
- Safer production caching and cache refresh behavior
- Improved maintainability through central configuration and versioned assets

## Deployment
Preview the complete folder using Live Server. After approval, replace the repository contents with this release, commit, push, and verify all pages on the deployed domain.
