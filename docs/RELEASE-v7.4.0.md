# Resplendent Global Travel Solutions v7.4.0

## Search & Performance Edition

This release strengthens search visibility, page delivery and production readiness without changing the approved design or editorial content.

### Included
- Unique titles and descriptions across all public pages
- Canonical, Open Graph and Twitter metadata
- Consolidated Organization, TravelAgency, WebSite, WebPage and breadcrumb structured data
- High-quality WebP image alternatives while retaining original JPEG files
- Critical homepage hero preload and lazy loading for destination images
- Apache compression, caching, HTTPS canonicalization and security headers
- Updated sitemap and robots directives
- Versioned CSS and JavaScript references

### Deployment note
Copy the release contents into the existing Git-connected project folder. The package intentionally excludes `.git`. Preview locally, commit, push to GitHub, then upload the same release files to Verpex.
