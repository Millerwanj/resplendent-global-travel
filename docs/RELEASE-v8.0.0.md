# Release v8.1.0 — Clean Editorial Upgrade

## Included
- Removed Payments from public navigation while retaining the private page.
- Introduced extensionless canonical URLs with permanent redirects from legacy `.html` URLs.
- Updated internal links, sitemap, canonical URLs, Open Graph URLs and structured data.
